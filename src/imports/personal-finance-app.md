Design a modern **Personal Finance Tracker Web Application** intended to be implemented using a **React-based frontend**.

The application helps users track their income, expenses, savings, and financial habits. The UI should follow a **component-based layout suitable for React development**, where each section can easily be converted into reusable React components.

Use a **modern fintech-style dashboard design** with a clean layout, soft colors, rounded cards, clear typography, and a professional SaaS-style interface.

All financial values must use **Indian Rupees (₹ INR)** and follow **Indian number formatting** (example: ₹1,25,000 instead of 125000).

Create the following pages:

1. Login / Signup Page
   A clean authentication page with:

* Email input
* Password input
* Login button
* Create account option
* Finance-related illustration
* Minimal modern design

2. Dashboard Page
   A financial overview dashboard containing:

* Total Balance card
* Monthly Income card
* Monthly Expenses card
* Savings card

Example values:
Total Balance: ₹45,000
Income: ₹25,000
Expenses: ₹12,500
Savings: ₹7,500

Also include:

* Pie chart showing expense categories
* Bar chart showing monthly spending
* Recent transactions list
* Quick “Add Transaction” button

3. Add Transaction Page
   A form to add new financial entries with fields:

* Transaction type (Income / Expense)
* Amount (₹ INR)
* Category
* Date
* Notes
* Save button

Categories should include:
Food
Transport
Rent
Groceries
Shopping
Bills
Entertainment
Medical
Others

4. Transactions History Page
   A table showing all transactions including:

* Date
* Category
* Amount
* Type (Income / Expense)

Include:

* Search bar
* Filters by category and date
* Edit and delete actions

5. Analytics Page
   A financial insights page showing:

* Expense distribution pie chart
* Monthly spending bar chart
* Category spending breakdown
* Savings trend visualization

6. Budget Management Page
   Allow users to set monthly budgets for categories.
   Show progress bars indicating spending vs budget.

Example:
Food Budget: ₹5,000
Spent: ₹3,200

7. Layout Structure
   Use a professional dashboard layout including:

* Left sidebar navigation (Dashboard, Transactions, Analytics, Budgets, Settings)
* Top navigation bar
* Card-based UI components
* Charts and visual analytics
* Responsive desktop layout

The design should be structured so that it can easily be converted into **React components** such as Dashboard, TransactionList, AddTransactionForm, AnalyticsCharts, and BudgetTracker.

Style inspiration: modern fintech dashboard with clean spacing, clear data visualization, and user-friendly navigation.
