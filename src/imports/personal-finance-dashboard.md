Design a modern web application UI for a **Personal Finance Tracker Dashboard**.

The application helps users track their income, expenses, savings, and financial habits.

Use a **clean, modern SaaS-style design** with soft colors, rounded cards, and a minimal dashboard layout. The interface should look professional and suitable for a fintech product.

Create the following pages:

1. **Login / Signup Page**

* Simple authentication form
* Email and password fields
* “Create Account” and “Login” buttons
* Illustration related to finance or money tracking

2. **Main Dashboard**
   Show a financial overview using cards and charts:

* Total Balance
* Monthly Income
* Monthly Expenses
* Savings

Include:

* Expense breakdown pie chart
* Monthly spending bar chart
* Recent transactions list
* Quick “Add Expense” button

3. **Add Transaction Page**
   Form to add:

* Transaction type (Income / Expense)
* Amount
* Category (Food, Transport, Shopping, Bills, Entertainment, Others)
* Date
* Notes

4. **Transactions History Page**

* Table showing all transactions
* Filters by category and date
* Search bar
* Edit and delete options

5. **Analytics Page**
   Visual insights including:

* Expense distribution by category
* Monthly spending trends
* Budget vs actual spending

6. **Budget Management Page**

* Users set monthly budgets for categories
* Progress bars showing spending vs budget

Design requirements:

* Sidebar navigation (Dashboard, Transactions, Analytics, Budgets, Settings)
* Responsive layout for desktop
* Use charts, cards, and modern UI components
* Professional fintech-style interface
